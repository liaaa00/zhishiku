# 分公司统计最终口径确认

## 结论

针对「北仑分公司开设最新进度表0310.xlsx」中“目前北仑在哪些城市开了分公司 / 多少个城市开设了分公司”一类统计问题，最终采用 **36** 作为标准答案。

## 判定规则

- 以 `当前进度-4.开设公司名称`（语义列：`branch_company`）为主判定依据。
- 该字段为有效名称即计入；有效名称指非空，且不是 `未开设`、`无`、`暂无`、`否` 等否定/占位值。
- 不额外要求公司名称必须包含“分公司”三个字。
- 不因问题中出现“开设了/开了分公司”就自动叠加银行账户、社保公积金账户均完成的过滤条件。
- 如果用户显式要求“银行和社保都办好/开具完成”，才叠加对应完成度过滤。

## 宁波北仑条目

- 城市：`宁波北仑`
- 开设公司名称：`外服（浙江）企业服务有限公司`
- 虽然名称不含“分公司”，但名称栏为有效公司名称，因此按最终口径计入。

## 回归要求

- 代码中不得用“名称必须包含分公司”替代上述显式口径。
- 检索评测只验证路由与证据命中，不验证最终答案文字；分公司数量这类聚合口径必须用单元测试和人工口径文档共同约束。
- 修改表格检索、筛选解析、行过滤逻辑后，必须保持统一检索评测基线 **24/24**。
